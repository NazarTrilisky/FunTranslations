Fun Translations

This is a web application to translate input text to character speak.  For example, if the input text is "I crossed the seas" and the character is "pirate", then the output could be "Yarr mateys, I crossd' ther seas."  A screenshot of the website is attached.

Installation:
1. Install Python 3
2. Run "cd <directory_where_server.py_is>"
3. Run "python3 server.py"
4. Browse to http://localhost:47290
